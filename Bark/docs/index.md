# Welcome to Bark

This is a lightweight Markdown documentation server built with .NET 10.

## Features

- **Markdown parsing** - Full CommonMark and GitHub Flavored Markdown.
- **Hot reload** - Automatically updates when you change a `.md` file.
- **Navigation** - Generated from folder structure.
- **Table of Contents** - Automatically extracted from headings.
- **Previous/Next** - Pagination through pages.
- **Dark mode** - Respects system preference.
